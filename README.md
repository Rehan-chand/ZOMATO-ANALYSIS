# 🍽️ Zomato Analysis Dashboard

An interactive **Zomato Restaurant Analysis Dashboard** built to analyze restaurant trends, ratings, cuisines, costs, booking availability, online delivery, and geographical distribution of restaurants across multiple countries and cities.

This project transforms raw restaurant data into meaningful business insights using dashboards and visual analytics.

---

## 📌 Project Overview

The dashboard helps users analyze:

- Restaurant distribution across countries and cities
- Top cuisines by restaurant count
- Average ratings analysis
- Online delivery availability
- Table booking trends
- Cost for two analysis
- Restaurant opening trends over years
- Cuisine category insights
- Top food cities comparison

## 📈 Key Insights

### Restaurant Distribution
- Restaurants spread across multiple countries and cities
- India contains the highest restaurant concentration

### Customer Experience
- Most restaurants provide online delivery
- Table booking availability is comparatively lower

### Ratings Analysis
- Majority of restaurants fall into medium to high rating categories

### Pricing Analysis
- Most restaurants are concentrated in affordable price ranges

### Cuisine Trends
- North Indian cuisine appears among the most common categories
